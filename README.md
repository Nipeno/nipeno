# Nico

Ten years of tinkering turned into a structured approach to building. I'm a developer from Germany — self-taught first, now also studying Intelligent Systems Engineering — working across web apps, game tooling, and anything else that pulls at my attention.

## What I build

My current focus is production-ready frontend and full-stack web apps: React 19 + TypeScript, shipped via Vercel. I build with care for performance, clean code, and real deployment — not just local demos.

**NightZoom Tracker** is the project I'm actively developing: a real-time web dashboard for a FiveM server that surfaces live player counts, server stats, and world data. The stack is Vite + React + TypeScript on the frontend, with an Azure-hosted proxy layer that handles upstream API calls without exposing credentials to the client. Deployed on Vercel with analytics and speed monitoring integrated.

Beyond web work, I have years of Lua and C# experience from game modding and tooling, and occasional 3D work in Blender and Unity.
